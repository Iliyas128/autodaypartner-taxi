# AutoDayPartner

Project scaffold for AutoDayPartner — Vite + React + TypeScript + Tailwind.

## How to run locally

Install dependencies and start the dev server:

```powershell
npm install
npm run dev
```

Open http://localhost:5173 (or the URL printed by Vite).

## Technologies

- Vite
- React
- TypeScript
- Tailwind CSS
